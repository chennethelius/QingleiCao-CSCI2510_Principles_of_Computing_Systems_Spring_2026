3. function signatures

printf:
int printf(const char *format, ...);

fprintf:
int fprintf(FILE *stream, const char *format, ...);

fprintf() takes an additional first parameter (FILE *stream) that specifies
the output stream. printf() always writes to standard output (stdout),
while fprintf() allows you to specify it

4. description differences

The manual says printf() writes output to standard output.
fprintf() writes output to the specified output stream.
Therefore, fprintf() is more flexible because it allows writing to
different streams
